### Tecnologías usadas
 
Desarrollarás un sistema de recomendaciones con base en algoritmos de clustering para Machine Learning, optimizando la ingesta del modelo analizando los principales componentes de los insumos con algoritmos de reducción de la dimensionalidad y procesamiento de lenguaje natural.

* [Microsoft Edge 100.0.1185.39](https://www.microsoft.com/es-es/edge)
* [Google Colab](https://colab.research.google.com/)
* [Python 3.7.13](https://www.python.org/)
* [pandas 1.3.5](https://pandas.pydata.org/pandas-docs/version/1.3/getting_started/install.html)
* [numpy 1.21.5](https://numpy.org/devdocs/release/1.21.5-notes.html)
* [matplotlib 3.2.2](https://matplotlib.org/3.2.2/users/installing.html)
* [seaborn 0.11.2](https://seaborn.pydata.org/installing.html)
